亚马逊物流的功能、服务和费用

如果您加入了亚马逊物流，则可以借助可选功能和服务来帮助您管理和发展业务。亚马逊物流费用涵盖以下各项服务的费用：将您的商品存放在亚马逊运营中心；分拣、包装和配送订单；以及为所售商品提供客户服务。

亚马逊物流功能

针对亚马逊物流订单的客户服务： 如果您通过亚马逊物流销售商品，亚马逊将代表您提供客户服务。

全球开店： 通过亚马逊物流出口和亚马逊全球开店在全球范围内拓展您的业务。

亚马逊物流服务

亚马逊合作承运人计划： 亚马逊合作承运人可在将您的库存运输到运营中心时提供一定的折扣。

多渠道配送： 让亚马逊配送您通过自己的网站和其他渠道销售的库存订单。

亚马逊物流贴标服务： 亚马逊将为您的库存贴上条形码标签。此项服务按件收取费用。

亚马逊物流预处理服务： 亚马逊将对您的库存进行预处理，以使其符合亚马逊物流预处理要求。此项服务按件收取费用。

亚马逊物流重新包装服务： 亚马逊将对买家退回的符合要求的亚马逊物流商品进行重新包装，以便可以再次销售。

库存配置服务： 在您创建入库计划时，您的货件可能会被拆分为多个货件并发往不同的运营中心。您可以利用库存配置服务将所有库存发往一个运营中心，然后亚马逊会为您分发库存。此项服务按件收取费用。

人工处理服务： 如果您在将库存发往亚马逊物流时未提供箱内物品信息，亚马逊将在运营中心手动处理您的箱子。此项服务按件收取费用。

扫码贴标： 您可以通过“扫码贴标”使用条形码扫描枪扫描商品，然后逐个打印并粘贴标签，可更加轻松地确保将正确的标签粘贴到相应的商品上。

亚马逊物流费用

亚马逊物流订单的配送费用： 亚马逊物流的订单配送费用和相关服务费用概览。1

月度库存仓储费： 对于您存放在亚马逊运营中心的所有商品，亚马逊将按日历月和您的平均每日占用体积收取仓储费。

长期仓储费： 除月度仓储费外，亚马逊还会对亚马逊运营中心的库存评估收取长期仓储费。

移除订单费用和弃置订单费用： 您可以让亚马逊退还或弃置您存放在亚马逊运营中心的库存。此项服务按件收取费用。

退货处理费： 亚马逊将对提供免费退货配送的订单收取退货处理费。

计划外服务费用： 如果库存抵达运营中心时未经过适当的预处理或贴标，亚马逊将为您提供这些服务。此项服务按件收取费用。

亚马逊物流费用工具

费用估算工具

收入计算器 根据配送渠道估算费用和利润。（访问工具）
费用预览报告 下载一份报告，其中可显示您当前商品预计的“我要开店”费用和配送费用。（可通过配送报告找到）
“每件售出商品的预计费用”小部件 可预览售出某一 SKU 的一件商品时“我要开店”和“亚马逊物流”的主要费用。（可通过管理库存找到）
销售佣金预览报告 下载一份报告，其中可提供基于您当前发布的商品价格的预计销售佣金。（可通过库存报告找到）
GetMyFeesEstimate API 通过 GetMyFeesEstimate API 获取商品列表，并得到这些商品的“我要开店”费用和配送费用。

查看已支付费用的工具

亚马逊所收费用价目表 查看某个 SKU 在某段时间收取的费用。（可在 SKU 中心内通过点击管理库存中的 SKU 找到）
付款 本页面可提供付款一览信息，包括支付的费用。链接的报告可提供针对具体交易和事件支付的详细费用明细。（访问页面）
费用说明 请参阅“费用说明”，了解有关如何计算具体交易销售费用的更多详情。（可通过付款-交易一览，在查看某一交易详情时找到）
月度仓储费报告 下载一份报告，其中可显示您储存在亚马逊运营中心的库存中每个 ASIN 的预计月度仓储费。（可通过配送报告找到）
长期仓储费报告 下载一份报告，其中可针对储存在亚马逊运营中心的库存提供长期仓储费明细。（可通过配送报告找到）
库存仓储超量费报告 下载一份报告，其中可显示亚马逊运营中心内超出仓储限制的库存（基于仓储类型）的预计库存仓储超量费。（可通过配送报告找到）

1如果适用，将添加锂电池费用和特殊处理费。
